## مشروع مسابقة التكنولوجيا الخضراء :

# اسم المشروع :
بلدنا اخضر

الوصف:
موقع تفاعلي عن **التكنولوجيا الخضراء** يركز على مشاريع الكهرباء والطاقة المستدامة.  
الموقع يحتوي على:
- صفحة تعريفية بالتكنولوجيا الخضراء.
- أقسام عن مشاريع الكهرباء والطاقة.
- آلة حاسبة لتحويلات الكهرباء (فولت، وات، أمبير).

# طريقة التشغيل :
1 - قم بفك الضغط الخاص بالمشروع عن طريق الضغط بالزر الايمن بالفأرة و النقر علي "Extract Here" او "الاستخراج هنا"
2 - سيظهر لديك الملف الخاص بالمشروع , قم بفتحه
3 - سيظهر لديك محتويات المشروع (استمارة التقدم للمسابقة و استمارة لجنة تقييم العمل المقدم و صور بطاقة ولي امر الطالب و صورة شهادة ميلاد الطالب و ملفات الموقع او المشروع)
4 - قم بفتح الملف المسمي "index.html" و افتحه في اي متصفح (Google Chrome , Firefox , Brave , Edge)
5 - سيظهر الموقع و المشروع امامك مفتوح في المتصفح


# لغات البرمجة المستخدمة :
Html
Css
Javascript
